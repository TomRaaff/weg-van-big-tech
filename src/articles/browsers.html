Browsers

Een van de makkelijkste stappen die je kan zetten is een andere browser gebruiken. Stap af van Chrome. Goeie alternatieven zijn:

  • Brave
  • Firefox *
  • Zen Browser (modernere versie van Firefox)
  • LibreWolf (privacy variant van Firefox)
  • Mullvad

* Er is recentelijk ophef ontstaan rondom Firefox. Zij hebben namelijk een regel uit hun privacy statement verwijderd waarin zij beloofden NOOIT een gebruiker's data te verwijderen. Het verwijderen van die regel heeft een deur open gezet zodat ze dat wel kunnen doen. Veel mensen zijn hun vertrouwen in FireFox hiermee kwijt geraakt

Je kan de browser natuurlijk gewoon installeren en gebruiken, maar vergeet je gegevens niet over te zetten. En dan bedoel ik: je bookmarks en je wachtwoorden.

Bookmarks overzetten

Bookmarks exporteren uit Chrome

  1. Open Chrome en ga naar Bookmarks -> Bookmarks manager
  2. Klik op de 3 puntjes rechts bovenin de hoek
  3. Export bookmarks (als een .html bestand)
Bookmarks importeren in Brave
  1. Open Brave en ga naar Bookmarks -> Bookmarks manager
  2. Klik op de 3 puntjes rechts bovenin de hoek
  3. Import bookmarks
  4. Selecteer het bestand wat je vanuit Chrome hebt gedownload
Bookmarks importeren in Firefox/Zen
  1. Open de browser en ga naar Bookmarks -> Bookmarks manager
  2. Er opent een nieuw window
  3. Klik op de 'boven/beneden' pijltjes en Import bookmarks from html

Passwords overzetten

Passwords exporteren uit Chrome

  1. Open Chrome
  2. Klik op de 3 puntjes rechts bovenin de hoek: Passwords & autofill -> Google password manager
  3. Settings -> Export passwords
Bookmarks importeren in Brave
  1. Open Brave
  2. Klik op de 3 puntjes rechts bovenin de hoek: Passwords & autofill -> Google password manager
  3. Settings -> Export passwords
  4. Klik op de link naar Select a csv-file
  5. BELANGRIJK: verwijder je passwords file nadat je hem hebt geïmporteerd!
Bookmarks importeren in Firefox/Zen
  1. Open de browser en klik op de drie puntjes rechtsbovenin passwords
  2. rechtsbovenin staan wederom drie stipjes met opties om passwords te importeren
  3. BELANGRIJK: verwijder je passwords file nadat je hem hebt geïmporteerd!

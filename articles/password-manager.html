Gebruik een password manager

Wist je dat, nog steeds, de meeste accounts worden gehackt omdat mensen voor verschillende accounts dezelfde (simpele) wachtwoorden gebruiken? Dit is makkelijk te voorkomen: gebruik een password manager.

Een password manager is een tool waarmee je heel makkelijk je wachtwoorden kan beheren. Passwordmanagers zijn wat mij betreft een no-brainer. Het is de meest gebruiksvriendelijke en veilige manier om met accounts en wachtwoorden om te gaan. Een extra bijkomend voordeel is dat je daarmee je accounts goed kan bijhouden en makkelijk kan zien wat je (uiteindelijk) weer wilt verwijderen.

Password managers zijn extreem gebruikersvriendelijk omdat ze browser extensies hebben die jouw wachtwoorden al invuld op het moment dat je een website bezoekt. Kortom, je hoeft je wachtwoorden niet eens meer in te vullen. Je kan gewoon direct op login drukken. Vergeet dus niet de bijbehoorende browserextensie te installeren.

Voor password managers geldt wel: kies een betrouwbare. En eentje die security en encryption heel serieus neemt. Deze dienst beheert potentieel AL je accounts.
